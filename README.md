# Requerimientos

Tener instalado node.js/npm, nvm, angular, git y en el PATH

## Pasos a instalar lo necesario
Pagina principal de node: nodejs.org descargar version LTS seguir los pasos, agregalo al PATH y verificar instalacion (node -v, npm -v)
Instalar nvm en nvm-windows se descarga y se descomprime y corroborar version (nvm version)
Instalar angular con npm ya instalado npm install -g @angular/cli y corroborar version e instalacion (ng version)

## Ejecucion
Despues de instalar los requerimientos y se haya clonado el proyecto con git, accede desde terminal a la carpeta generada en este caso Pagina_Hackaton
Despues ejecuta el comando para instalar dependencias, versiones, y todo lo necesario para la compilacion de la pagina (npm install)
Para finalizar cada que se quiera ejecutar la pagina se ejecuta el siguente comando (ng serve -o) la -o solo es para que abra una pesta√±a nueva.
